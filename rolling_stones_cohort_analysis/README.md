Data Analytics Projects

A collection of end-to-end data science and machine learning projects covering exploratory data analysis, unsupervised learning, predictive modeling, and business insights.
📁 Projects
[Rolling Stones Song Cohort Analysis](./rolling_stones_cohort_analysis/)
Clustering Rolling Stones songs from Spotify into meaningful cohorts based on audio features using PCA and K-Means clustering.

👥 [Employee Turnover Prediction](./employee_turnover_prediction/)
Predicting employee attrition using machine learning models and classifying employees into retention risk zones to help HR teams take proactive action.
Tech Stack

- **Python** · **Pandas** · **NumPy**
- **Matplotlib** · **Seaborn**
- **Scikit-learn** · **Imbalanced-learn**
- 
How to Use

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/data-analytics.git
   cd data-analytics
   ```
2. Navigate into any project folder
   ```bash
   cd rolling_stones_cohort_analysis
   # or
   cd employee_turnover_prediction
   ```
3. Install dependencies and open the notebook
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn imbalanced-learn
   jupyter notebook
   ```
> Each project folder contains its own `README.md` with detailed steps, methodology, and results.
